# Portfolio de projets – Dabira Koro Michel Renaud

Bienvenue sur mon dépôt Git regroupant différents projets académiques, personnels et techniques réalisés dans le cadre de ma formation en Génie Électrique et Informatique Industrielle.

## Contenu du dépôt

### Projet_SQL_BD
Projet de base de données SQL : conception, création et manipulation de bases de données relationnelles.

### Projet_addionneur-BCD_Proteus
Conception et simulation d'un additionneur BCD sous Proteus à partir de composants logiques élémentaires.

### Projet_crossword_c
Développement d'un générateur de mots croisés en langage C avec gestion automatique du placement des mots dans une grille.

### Projet_decodeur_morse_Arduino
Réalisation d'un décodeur Morse utilisant une carte Arduino.
